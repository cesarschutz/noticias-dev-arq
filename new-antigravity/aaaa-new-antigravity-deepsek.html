Microsoft corrige 163 CVEs no Patch Tuesday de abril — SharePoint zero-day explorado ativamente

O Patch Tuesday de abril de 2026 traz 163 correções, com destaque para CVE-2026-32201, uma falha de spoofing no SharePoint Server já sob exploração ativa, e CVE-2026-33824 (CVSS 9.8), um RCE no serviço Windows IKE. Arquitetos devem priorizar patching em servidores SharePoint expostos e, onde não for possível aplicar, bloquear UDP 500/4500 para reduzir o blast-radius.

# Segurança & IAM

RedSun: zero-day no Windows Defender permite restaurar arquivos maliciosos com privilégios elevados

Publicada em 15 de abril, a RedSun explora erro lógico no tratamento de arquivos marcados com cloud tag pela Windows Cloud Files API — em vez de quarentena, o Defender reescreve o arquivo detectado no local original com privilégios elevados. Atores maliciosos começaram a weaponizar a falha em 16 de abril e, até fim do mês, ainda não há patch oficial.
